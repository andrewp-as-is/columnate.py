```bash
$ [sudo] pip install columnate
```

#### Examples
```python
>>> import columnate
>>> matrix = [[308, 0, "com.apple.Finder"],[84509, 0, "com.apple.metadata.mdwrite"]]
>>> columnate.lists(matrix)
308    0  com.apple.Finder
84509  0  com.apple.metadata.mdwrite
```
